## Features

- Mode Defaut/Gelap
- Export Excel dan PDF
- CRUD di Data Produk
- Validasi Pada Login dan Register 
- Menu Manajemen
- Submenu Manajemen
- Edit Profil
- Ubah Password
- Role Akses Manajemen


## Tech Stack

**Client:** HTML, Boostrap (CSS Framework), Javascript

**Server:** Codeigniter 3
